GromovWitten.eisenstein_series — Method
 eisenstein_series(q::Union{QQMPolyRingElem, Vector{QQMPolyRingElem}}, order::Int,k::Int)

Return the expansion of the weight Eisenstein series k with fixed order. For a fixed order $m$, we compute $E_k = 1 - \frac{2 k}{ B_k} \sum_{d=1}^{m} \sigma_{k-1}(d) q^{2 d}$

julia> eisenstein_series(q,5,2)  #E2
-144*q^10 - 168*q^8 - 96*q^6 - 72*q^4 - 24*q^2 + 1

julia> eisenstein_series(q,5,4) #E4
30240*q^10 + 17520*q^8 + 6720*q^6 + 2160*q^4 + 240*q^2 + 1

GromovWitten.inv_sfunction — Method
inv_sfunction(z::QQMPolyRingElem,aa::Int64)

returns the inverse sfunction

\[ \frac{1}{S(z,aa)}=\frac{z}{2 Sinh(z/2)}= \sum_{n = 0}^{aa} \left( \left\{\begin{array}{ll} 1 & \text{if} && n = 1\\ - \frac{- 2^n (- 2 + 2^n)}{n!} B_n & (n > = 1 && (- 1 + n)\mod 2 = 1) \end{array}\right. \right) z^n \]

Where $B_n$ is Bernoulli number and ${aa} \rightarrow \infty$.

Examples

returns solution of the system $Ax=b$, where A is a matrix from homogeneous Eisenstein series $E_2, E_4, E_6$ and $b$ from the Feynman Integral $I(q)$ The solution is of the form (factor, coefficients) where coefficients is a vector of rationals numbers.
